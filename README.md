# Emotion Detection Web Application

This project is a Flask‑based web application that detects emotions from text input using an NLP API. It analyzes user‑provided text and returns emotion scores for anger, disgust, fear, joy, and sadness, along with the dominant emotion.

## Features
Detects emotions from any text input
Handles blank input gracefully with error messages
Web interface built with Flask
JSON responses for easy integration
